# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you believe you have found a security issue in Obsern, report it by email:

- `obsern.security@gmail.com`

Please include as much detail as possible, such as:

- affected component or feature
- steps to reproduce
- impact
- logs, screenshots, or proof of concept if available

I will review the report and respond as soon as I can.

## セキュリティ報告（Japanese）

セキュリティ上の問題は、公開 Issue ではなくメールでご連絡ください。

報告先: `obsern.security@gmail.com`

可能であれば、対象箇所、再現手順、影響範囲、ログやスクリーンショットもあわせてお知らせください。
