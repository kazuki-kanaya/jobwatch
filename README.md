# Jobwatch


## Prerequisites

- Python (uv)
- Node.js (pnpm)
- Go (goenv)

## Dependencies

### Task

**macOS**

```bash
brew install go-task/tap/go-task
```

**Ubuntu / Debian**

```bash
curl -1sLf 'https://dl.cloudsmith.io/public/task/task/setup.deb.sh' | sudo -E bash
sudo apt install task
```

### Lefthook

```bash
go install github.com/evilmartians/lefthook@latest
```

```bash
lefthook install
```
