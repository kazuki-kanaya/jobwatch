---
title: Getting Started
description: Fast setup flow for Obsern.
---

## 1. Install CLI

```bash
brew tap kazuki-kanaya/obsern && brew install obsern
```

## 2. Initialize config

```bash
obsern init
```

## 3. Wrap your command

```bash
obsern run python train_model.py --epochs 100
```

You can check completion/failure from dashboard and notifications.
