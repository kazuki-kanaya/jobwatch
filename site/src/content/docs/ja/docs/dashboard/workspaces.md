ワークスペースは、ホスト・ジョブ・メンバーを管理する単位です。

## 重要ポイント

- ワークスペースは複数作成可能
- ホストは1ワークスペースに紐づく
- メンバー権限はワークスペースごとに管理

## 作成手順

1. Dashboard を開く
2. ワークスペース領域の `+ Add` を押す
3. 名前を入力して作成
