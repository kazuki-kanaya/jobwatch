## ロール

- `viewer`: 閲覧のみ
- `editor`: viewer + ホスト作成/削除 + ジョブ削除
- `owner`: editor + ワークスペース/メンバー管理

## owner の追加権限

- ワークスペース名変更/削除
- 招待リンク作成
- メンバー追加/除名
- 権限変更
- owner 権限委譲

## owner のルール

- ワークスペース作成者が owner
- 権限変更で owner は複数可
