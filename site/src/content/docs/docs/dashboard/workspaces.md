A workspace is a management scope for hosts, jobs, and members.

## Key Points

- You can create multiple workspaces.
- Hosts belong to a single workspace.
- Members and roles are managed per workspace.

## Create Workspace

1. Open dashboard.
2. Click `+ Add` in the workspace area.
3. Enter workspace name and create.
