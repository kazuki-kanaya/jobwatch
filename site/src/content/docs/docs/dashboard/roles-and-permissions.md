## Roles

- `viewer`: read-only.
- `editor`: viewer + host create/delete + job delete.
- `owner`: editor + workspace/member administration.

## Owner Can Also

- Rename/delete workspace
- Create invitation links
- Add/remove members
- Change member roles
- Transfer owner role

## Ownership Rules

- Workspace creator becomes owner.
- Multiple owners are allowed after role updates.
