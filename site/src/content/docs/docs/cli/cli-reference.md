---
title: CLI Reference
description: Main Obsern CLI commands.
---

Common commands:

```bash
obsern --version (or: obsern version)
obsern init
obsern run <command>
obsern --help
```

Usage flow:

1. Run `obsern init` to generate `obsern.yaml`.
2. Add required notification/auth settings.
3. Run jobs via `obsern run <command>`.
